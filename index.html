Hvordan sette opp en server

En server er en spesialisert datamaskin som har alt sin behandling og harddisk resurser dedikert til en bestemt funksjon, som den fullfører mange ganger. Du kan bruke en server til å regulere Internett-trafikk mellom alle datamaskinene på nettverket.

I denne bruksanvisning får du vite hvordan du setter opp Ubuntu 20.04, uten å ha noe kunnskap om det fra før av.

Steg 1) Bygg sammen serveren.

  • Åpner kabinettet.
  • Sett inn Hovedkortet
  • GPU sitter i hovedkortet.
  • På hovedkortet fester du:
  • RAM, CPU, CPU kjøler, SSD, PSU.

Steg 2) Sett opp en minnepinne med Ubuntu server.

  • Du setter en minnepinne i pc-en
  • Du bruker rufus (det må man laste ned)
  • Laster ned fra denne nettsiden: Ubuntu
  • Så overfører du det du lasta ned, til minnepinnen.
  • Etter det tar du minnepinnen i serveren.
  • Da burde du få opp dette.

Sette opp selve grunninnstillingene på serveren

  • Hvis serveren din er koblet til nett, får du denne siden opp. Hvis ikke hopper den over dette. Men hvis du får opp denne siden velger du “update to the new installer”
  • Etter dette kommer du til å velge språk og tastatur oppsett. Der kan du velge hvilket språk du vil.
  • Etter du har valgt språk får du flere sider opp, hvor du trykker “done“/går videre.
  • Når du kommer til “proxy server”, er dette ikke nødvendig å ha med. så du går bare vider uten å fylle ut noe.
  • Trykk “done” igjen, helt til at du kommer hit:
  • Når du kommer til denne siden kan du lage dit eget navn, server navn, brukernavn og passord. Dette må du huske på, så det er lurt å velge noe som er lett å huske.
  • Trykk “done“.

steg 3) Installerer SSH

  • Når du kommer til denne siden, velger du å installere SSH, det gjør Du ved å sette kryss som vist på bilde.
  • Trykk “done“.
  • Denne siden hopper du bare over.
  • Bare gå videre uten å trykke noe.

Nå vil den installere, vent til installasjonen er ferdig.

Velg “Reboot”. Nå er installasjonen ferdig.

steg 4) Sette opp statisk IP

Nå skal du sette opp statisk IP, se denne videoen.

  • Først må du finne IP-en din. da skriver du “ip a“.
  • Da burde noe som dette komme opp.
  • Hvor det er outlined er viktig å merke seg. (den er forskjellig for alle).
  • Etter det skriver du inn koden 
 sudo nano /etc/cloud/cloud.cfg.d/subiquity-disable-cloutinit-networking.cfg

eEtter du har skrevet inn den koden kommer det opp^

Pass på at det står “disabled“

sudo nano /etc/netplan/00-installer-config.yaml

Da kommer noe som dette. Du skal bytte til hva du selv har.

Lagre koden

For å teste om koden virker skriver du denne koden.

sudo netplan apply

Og så for å starte på nytt, skriv denne koden:

sudo reboot

Nå har du statisk IP og kan gå videre til brannmur

Steg 5) Sette opp brannmur

Først så kan du sjekke hvis du allerede har brannmur med denne koden:

sudo ufw status

Denne koden er for å muliggjøre 

sudo ufw enable

Nå burde brannmur være innstalert.

Steg 6) Sette opp Apache

Nå installerer vi Apache. første koden er:

sudo apt-get install apache2

Trykk “Y” og ENTER for å tillate installasjonen.

Sjekk om Apache er installert riktig med denne koden:

sudo service apache2 status

Dette er hva du burde se hvis det ble gjort riktig.

  • FOR Å KOMME SEG UT TRYKK “Q”

For å sjekke om brannmuren virker med Apache skriver du:

sudo ufw app list

For å sikre at Apache kjører, skriv inn IP-adressen til serveren din i adressefeltet og trykk ENTER. da burde det se sån ut:

Steg 7) Sette opp MySQL

Først så må vi lage en database

Skriv denne koden:

sudo apt-get install mysql-server

Trykk på “Y” for å godkjenne installasjonen 

mysql_secure_installation

Trykk “Y” for å lagre når det kommer opp.

etter du har installert skal du lage en bruker, da skriver du først denne kommandoen.

sudo mysql

skriv dette med dit brukernavn (username). Du skal fortsatt ha “@localhost” bak.

mysql> CREATE USER username@localhost IDENTIFIED BY password;

Nå skal du gi tilgang til den brukeren du laget.

mysql> GRANT ALL PRIVILEGES ON *.* TO username@localhost WITH GRANT OPTION;

skriv denne koden for å lagre

mysql> FLUSH PRIVILEGES;

nå kan du exite MySQL.

Steg 8) Sette opp PHP

For å installere PHP skriver du denne kommandoen. 

sudo apt-get install php libapache2-mod-php php-mysql

Press “Y” for å tillate installasjonen.

Etter bør du endre måten Apache sender filer på. Som standard ser Apache først etter index.html. Vi vil for nå at den skal se etter index.php i stedet.

For å bytte dette, åpner du dir.conf, med root privilegier. med denne kommandoen:

sudo nano /etc/apache2/mods-enabled/dir.conf

Da ser du dette:

Du skal velge index.php.

Trykk CTRL + X for å lagre. Og “y” for å bekrefte

Restart Apache 

sudo systemctl restart apache2

Neste er WordPress.

Steg 9) sette opp wordpress

For å installere wordpress bruker du først denne kommandoen:

sudo mkdir -p /srv/www
sudo chown www-data: /srv/www
curl https://wordpress.org/latest.tar.gz | sudo -u www-data tar zx -C /srv/www

Neste kommando er:

<VirtualHost *:80>
    DocumentRoot /srv/www/wordpress
    <Directory /srv/www/wordpress>
        Options FollowSymLinks
        AllowOverride Limit Options FileInfo
        DirectoryIndex index.php
        Require all granted
    </Directory>
    <Directory /srv/www/wordpress/wp-content>
        Options FollowSymLinks
        Require all granted
    </Directory>
</VirtualHost>

For å muliggjøre siden med:

sudo a2ensite wordpress

Aktiver URL-en med:

sudo a2enmod rewrite

Deaktiver standard nettstedet med:

sudo a2dissite 000-default

Nå laster du inn Apache på nytt for å bruke alle disse endringene:

sudo service apache2 reload

Nå skriver du inn DIN IP-adresse i adressefeltet + /wordpress

Da burde du komme til noe som dette:

her skriver du inn: nettside navnet, brukernavn, passord, database vert.

her skriver du inn hva du vil siden skal hete, brukernavn, passord og E-mail.

Steg 10) Sette opp FTP

for å installere FTP starter vi med å bare oppdatere:

sudo apt update

neste er å installere vsftpd med denne kommandoen: 

sudo apt install vsftpd

dette er sån ca hvordan det skal se ut:

for å starte og aktivere, skriv disse kommandoene:

sudo systemctl start vsftpd

sudo systemctl enable vsftpd

Nå lager vi en sikkerhetskopi med denne kommandoen:

sudo cp /etc/vsftpd.conf  /etc/vsftpd.conf_default

Nå kan du lage en FTP bruker, det gjør du med følgene kommandoer:

sudo useradd -m testuser

sudo passwd testuser

systemet burde spørre om å lage et passord.

for å passe på at brannmuren ikke blokker FTP, skriver du disse kommandoene:

sudo ufw allow 20/tcp

sudo ufw allow 21/tcp

dette er sån ca hva du burde få opp.

Neste er å koble til Ubuntu FTP server. Da skriver du denne kommandoen: 

sudo ftp ubuntu-ftp

Bytt “ubuntu-ftp” med navnet på systemet ditt. Det tar du fra kommando linjen.

Etter logger du inn med det passordet og brukernavnet du lagde. Da burde du være logget inn på FTP.

Nå kan du sikre Ubuntu vsftpd serveren, med å endre standard katalogen. Det gjør du med denne kommandoen: 

sudo mkdir /srv/ftp/new_location

sudo usermod -d /srv/ftp/new_location ftp

Restart vsftpd for å ta i bruk endringene.

Hvis du vil laste opp filer rediger vsftpd.conf med å skrive disse kommandoen:

sudo nano /etc/vsftpd.conf

skift “write_enable=NO” og skift det til “YES“

Lagre filen, exit og restart FTP med denne kommandoen:

sudo systemctl restart vsftpd.service

Skriv denne kommandoen:

chroot_local_user=YES

Nå endrer du i vsftpd.conf med disse kommandoene:

chroot_local_user=YES

chroot_list_file=/etc/vsftpd.chroot_list

Restart vsftpd med denne koden:

sudo systemctl restart vsftpd.service

Neste åpner du vsftpd.conf og bytt ssl_enable=NO til ssl_enable=YES.

ssl_enable=YES

Nå har du installert alle tingene du trenger inkudert LAMP, SSH, statisk IP, brannmur, wordpress og FTP.
